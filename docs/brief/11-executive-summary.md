# AURA HQ — Executive Summary
**One-page reference | April 2026**

---

## What it is

A personal AI infrastructure platform. One person — the Principal — sits at the apex of a visual organizational hierarchy. Beneath them: scoped AI agents, each with a defined domain, user-written purpose, domain-specific capabilities, and persistent memory. The product has three surfaces: HQ (the structure), Workspace (the desk), Creation Flow (the ceremony of building).

**Core promise:** *For the first time, AI has a structure — and you are at the top of it.*

---

## Target user

Intelligent adult — founder, creative, knowledge worker — who has used ChatGPT and found it useful but impersonal. Not a developer. Not an AI enthusiast. Someone who would pay for something excellent that genuinely works for them.

---

## The three surfaces

| Surface | Purpose | Navigation |
|---|---|---|
| HQ | Structure at a glance. Principal at apex, agents below, ambient status. | Swipe up from any workspace to reach. |
| Workspace | Work with one agent. Capability strip, output cards, memory context. | Swipe left/right between agents. |
| Creation Flow | Make a new agent. 5-step ritual. Domain, name, purpose, seeding, placement. | Opens from empty node. Closes to HQ. |

Everything else is a panel within one of these three.

---

## The visual identity in one line

The cold geometry of institutional authority — inhabited by the warmth of one person.

**Primary palette:** Burnt Umber Dark `#16100A` · Antler `#EFE8DA` · Resin Amber `#B8762A` (Principal only, ≤3× per surface) · Burnished Sienna `#5C4A38` · Deep Slate Warm `#2A2620`

**Typography:** Canela/Lyon Text (structural authority only) + Söhne/ABC Diatype (all operational copy)

---

## MVP ships

3 domains (Work / Finance / Health) · Full onboarding ceremony with avatar generation · HQ view up to 4 agents · Agent creation flow (seeding required) · Workspace with capability strip + output cards + memory · Spatial navigation (swipe up/horizontal) · Flat monthly subscription ~$25, 14-day trial

## MVP does not ship

Domains 4–6 · Discovery screen · Weekly synthesis · Mobile app · Sharing · Push notifications · Second-tier HQ · Cross-agent intelligence

---

## Three things that must be true at launch

1. **The avatar is consistently excellent** — passes the "nice portrait photo" test across diverse inputs. If not: launch-blocker.
2. **Seeded agents produce visibly better first outputs than generic AI** — the delta must be unmistakable in session one.
3. **The HQ delivers specific, actionable agent state on every return** — names real agents with real states. If data isn't ready: ship node visual states only, no text.

---

## Design mantra

**Does this make the user feel like the principal of their own intelligent world, or like a user of someone else's product?**

Principal → keep it. User → cut it.
